Angine.

A virtual audio mixer that allows users to set up, manage, and save audio settings.

A course project inspired from my frustration when setting audio for my home studio.

Angine is a course project to design and build a marketable product. My frustration inspired the project of setting up my audio system when I was trying to set up a home studio. As a tribute to my pain, I designed a virtual audio mixer that optimized setting up, managing, and saving audio settings to handle multiple audio devices and tasks.
